Installing Node.js

First, you must download and install node.js. NPM stands for node packaged modules and is a way to manage development dependencies through Node.js.

Download the Node.js source code or a pre-built installer for your platform, and start developing, you can download it from nodejs.org

You can check it in your terminal window using these commands node --version and yarn --version.

Installing Gulp

GulpJS is a JavaScipt task runner. In one word: automation. The less work you have to do when performing repetitive tasks like minification, compilation, unit testing, linting, etc...

Vuexy Build System use the gulp task to generate the different templates.

From the command line:

  • Install gulp globally with yarn add gulp-cli -g
  • To install node packages, navigate to the root vuexy-html-bootstrap-admin-template/directory. Then run yarn. Yarn use the package.json file and automatically install the required local dependencies listed in it.

Gulp Commands

gulpfile.js file contain all the predefined task. Below template contain all the gulp task and usage.

You can modify or add your task in gulpfile.js file.

Command Description
gulp dist Gulp task to generate css and js files in app-assets folder.
gulp monitor Watch all scss and compile it accordingly. In this command you need to pass the Layout, LayoutName & TextDirection.
gulp sass-compile Compile core, main(app), pages and plugins scss files.
gulp dist-css Clean css folder, compile all scss files, auto prefix them, organize them and finally minify them in app-assets/css folder.
gulp dist-js Gulp task to clean js folder from app-assets, copy js files from src folder and minify them.
gulp dist-vendor-js Gulp task to generate combined vendor js file in app-assets/vendors/js/vendors.min.js. It includes all required JS vendors for this template. Use config.json file vendors object to manage vendors.
gulp dist-vendor-css Gulp task to generate combined vendor CSS file in app-assets/vendors/css/vendors.min.css. It includes all required CSS vendors for this template. Use config.json file vendors object to manage vendors.
gulp dist-clean Clean js and css folders from app-assets.
