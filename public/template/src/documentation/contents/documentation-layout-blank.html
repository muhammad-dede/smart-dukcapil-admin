CSS Classes

This table contains all classes related to the layout blank. This is a custom layout classes for blank layout page requirements.

All these options can be set via following classes:

Classes / Attribute Description
.blank-page You can create blank layout by adding .blank-page class with <body> tag.
data-col="blank-page" You can create blank layout by adding .data-col="blank-page" attribute with <body> tag.

HTML Markup

This section contains HTML Markup to create layout blank. You need to add .blank-page class as well as data-col="blank-page" attribute to <body> tag and also remove <div> section with .main-menu menu-fixed.menu-light.menu-accordion.menu-shadow classes, <nav> section with header-navbar.navbar.navbar-expand-lg.align-items-center.floating-nav.navbar-light.navbar-shadow classes and <footer> section with .footer.footer-static.footer-light classes. Default layout of the template comes with menu as well as navbar but you can create blank layout with below changes.

Vuexy has a ready to use starter kit, you can use this layout directly by using layout-blank.html. 


<!DOCTYPE html>
<html lang="en">
  <head>
    ...
  </head>
  <body class="vertical-layout vertical-menu-modern navbar-floating footer-static blank-page" data-menu="vertical-menu-modern" data-col="blank-page">   <!-- Add here -->
    <!-- BEGIN: Content -->
    <div class="app-content content">
      <div class="content-wrapper">
        ...
      </div>
    </div>
    <!-- END: Content -->
  </body>
</html>
