HTML Markup

This section contains HTML Markup to create layout without menu. You need to add data-col="1-column" attribute to <body> tag and also remove <div> section with .main-menu menu-fixed.menu-light.menu-accordion.menu-shadow classes. Default layout of the template comes with menu but you can create without menu layout with below changes.

Vuexy has a ready to use starter kit, you can use this layout directly by using layout-without-menu.html. 


<!DOCTYPE html>
<html lang="en">
<head>
  ...
</head>
<body class="vertical-layout vertical-menu-modern navbar-floating footer-static" data-menu="vertical-menu-modern" data-col="1-column">   <!-- Add here -->

  <!-- BEGIN: Navbar -->
  <nav class="header-navbar navbar navbar-expand-lg align-items-center floating-nav navbar-light navbar-shadow">
    ...
  </nav>
  <!-- END: Navbar -->

  <!-- BEGIN: Content -->
  <div class="app-content content">

      <!-- Content-wrapper -->
      <div class="content-wrapper">
        ...
      </div>

  </div>
  <!-- END: Content -->

  <!-- Begin: Footer -->
  <footer class="footer footer-static footer-light">
    ...
  </footer>
  <!-- END: Footer -->

</body>
</html>
